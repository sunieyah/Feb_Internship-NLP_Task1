# NLP Preprocessing Engine

## 📌 Overview
This project implements an advanced NLP preprocessing pipeline as part of a Data Science Internship assignment.

## ⚙️ Features
- Lowercasing
- URL & email removal
- Number removal
- Repeated character normalization
- Special character removal
- Tokenization
- Stopword handling (custom rules)

## 🧪 Tasks Covered
- Conceptual Understanding
- Advanced Preprocessing Function
- Stress Testing
- Token Analytics
- Frequency Analysis
- Full Pipeline
- Error Handling

## 🛠️ Tech Used
- Python
- Regular Expressions (re)
- Collections (Counter)
